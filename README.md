#  BAMS - Blockchain Attendance Management System

## Overview
A secure, immutable attendance management system built on blockchain technology with a 3-layer hierarchical structure: Department ‚Üí Class ‚Üí Student ‚Üí Attendance.

##  Features
-  SHA-256 hashing with Proof of Work (difficulty 4)
-  Complete CRUD operations
-  Hierarchical blockchain linking
-  Real-time validation
-  Modern React UI with Tailwind CSS

##  Quick Start

### Backend
```bash
cd Backend
npm install
node server.js
```

### Frontend
```bash
cd Frontend
npm install
npm start
```



##  Tech Stack
- **Backend:** Node.js, Express.js, SHA-256 Crypto
- **Frontend:** React 18, Axios, Lucide Icons, Tailwind CSS
- **Storage:** JSON Files (MongoDB ready)

- **Backend:** [http://localhost:5000/api]
